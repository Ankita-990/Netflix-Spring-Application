# Netflix Content Management System

![image](https://github.com/Ankita-990/Netflix-Spring-Application/blob/main/src/main/resources/static/netflix_content_manager.png)

### 1. The Netflix Content Management System is a Java Spring based web application, mainly focusing on creating POST and GET API
### 2. The application is use to add, view, and search Netflix-style content through a browser interface that communicates directly with a REST API.
### 3. The frontend in compiled of HTML, CSS, and JavaScript
### 4. MVC structure is followed to seperate the business logic from entity
### 5. Array-based in-memory data storage is used
